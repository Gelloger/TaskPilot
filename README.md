# TaskPilot

Dooray 태스크 품질 관리 AI 코파일럿 — MVP 웹 Mock 프로토타입

## 개요

TaskPilot은 Dooray 프로젝트의 태스크 품질을 자동으로 분석하고, AI 초안 댓글로 보강을 도와주는 코파일럿 서비스입니다. 이 레포지토리는 UX 검증을 위한 인터랙티브 웹 mock입니다.

## 주요 화면

| 화면 | 설명 |
|------|------|
| **프로젝트 목록** | 연결된 Dooray 프로젝트별 품질 점수 현황 |
| **케어 레벨 설정** | 프로젝트별 품질 체크 기준(1~5단계) 설정 |
| **태스크 현황** | 봇 실행 → Before/After 품질 변화 데모 |
| **AI 초안 상세** | 봇이 작성한 초안 댓글 확인 및 반영 |
| **통계 대시보드** | 품질 분포, 담당자별 현황, 봇 도입 효과 |

## 데모 시나리오

1. **프로젝트 목록**에서 "NHN Cloud SMS 서비스" 카드 클릭
2. **태스크 현황**에서 낮은 품질 점수 확인 (Before)
3. **"TaskPilot 봇 실행"** 클릭 → 스캔 애니메이션 진행
4. 품질 바가 초록색으로 업데이트, 우하단 메신저 알림 팝업 (After)
5. 태스크 카드 클릭 → **AI 초안 댓글** 확인 → "반영" 버튼 클릭
6. **통계 대시보드**에서 봇 도입 전/후 품질 추이 확인

## 기술 스택

- **Vite 6** + **React 18**
- **Tailwind CSS v3**
- **React Router v6**
- **Recharts** (통계 차트)
- **Lucide React** (아이콘)
- 더미 데이터 기반 (Dooray API 연동 없음)
